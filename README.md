                                                    Schedule
                                                      v 1.0

这是一个基于windows phone SDK 7.1的课程表application。开发环境 windows 7 + vs 2010 + windows phone SDK 7.1.

## 功能描述：
  1. 可以实现课程的添加和删除
  2. 对课程的描述
  3. 定时提醒


在设计方面采用HubTile组件和全景视图开发，给用户一种全新的感觉。
